# 1. Objective
- Verify application functionality in an e-commerce application
- Verify user interface elements and usability. 
- Detection of defects during testing and before release. 
- Verify user security.
- Verify fixed defects (Regression Test) 

# 2. Scope
- Login
- Product page (Picture-description)
- Add/Remove Items from cart page
- Filter option
- Checkout page
- Remove Items from checkout page 
- Options on main page and Cart page
- Menu options
- Logout option

#Out of scope: 
- Payment functionality
- Elements integration

# 3. Test Strategy
- Functional testing
  Verify that the application functionality works according to the requirements
- UI Testing
  Verify buttons, text, pictures, layout and respondence
- Regression testing
  Verify that reported defects don't affect the functionality and or new changes. 
- Smoke Testing
  Verify the basic functionality of the application 
- Testing method
  Manual Testing
