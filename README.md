# QA-portfolio-saucedemo

## Project Description
Manual QA project on SauceDemo, focused on validating the e-commerce purchase flow through test case design and execution, defect identification, and results reporting.

## Scope
Functional testing of:
- Login
- Product listing
- Cart
- Checkout

## Testing Types
- Functional testing
- UI testing
- Exploratory testing
- Regression testing

## Tools Used
- Test case management: Excel
- Bug tracking: Jira style reports

## Test Environment
Browser: Chrome
OS: Windows
